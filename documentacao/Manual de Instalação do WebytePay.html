Manual de Instalação do WebytePay

Passo 1: Preparação do Ambiente

  • Verifique os Requisitos: Certifique-se de que seu ambiente de desenvolvimento atende aos requisitos do Vendi Aqui e do WebytePay.
  • Acesso ao Servidor: Tenha acesso ao servidor onde o Vendi Aqui está instalado.

Passo 2: Descompactar e Copiar o Plugin

  • Descompactar o Plugin: Descompacte o arquivo .zip do WebytePay que você criou.
  • Copiar para o Diretório: Copie a pasta webytepay para o diretório /extras/plugins/ do seu servidor.

Passo 3: Instalação pelo Painel de Controle

  • Acessar o Painel de Controle: Entre no painel de administração do Vendi Aqui.
  • Navegar para Plugins: Vá até a seção "Plugins".
  • Instalar o WebytePay: Clique em "Instalar" ao lado do WebytePay.

Passo 4: Configuração do Gateway

  • Configurar Métodos de Pagamento: Navegue até Admin panel → Payment → Payment Method.
  • Edite as configurações do WebytePay conforme necessário.

Passo 5: Testar o Funcionamento

  • Realizar um Pagamento de Teste: Tente realizar uma transação usando o WebytePay para garantir que tudo esteja funcionando corretamente.
  • Verificar o Registro de Pagamentos: Acesse Admin panel → Payments → List e verifique se a transação foi registrada.

Passo 6: Solução de Problemas

  • Se encontrar algum problema durante a instalação ou testes:
  • Verifique os logs do servidor para mensagens de erro.
  • Confirme se todas as dependências e configurações foram corretamente ajustadas.
  • Se necessário, revise o código do WebytePay para identificar possíveis erros.

Conclusão

Pronto! Agora o WebytePay deve estar instalado e funcionando corretamente no Vendi Aqui. Em caso de dúvidas, consulte o suporte técnico.
