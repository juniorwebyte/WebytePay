Log de Alterações - WebytePay

Versão 2.3.0 - 2024-11-01
Melhorias:
  • Otimização do carregamento do gateway de pagamento para reduzir o tempo de resposta.
  • Refatoração do código para facilitar futuras integrações com outros métodos de pagamento.
  • Melhoria na interface de usuário do formulário de pagamento para aumentar a clareza e acessibilidade.
Correções de Bugs:
  • Correção de erro que impedia a atualização automática de status após o pagamento.
  • Correção de incompatibilidade com PHP 8.1.
  • Correção de problema na exibição de mensagens de erro ao tentar realizar pagamentos com dados incompletos.
Versão 2.2.1 - 2024-10-15
Melhorias:
  • Aprimoramento de segurança na comunicação entre WebytePay e Vendi Aqui, utilizando criptografia adicional.
  • Adicionada opção para habilitar/desabilitar notificações via SMS no painel de administração.
Correções de Bugs:
  • Correção de problema no envio de notificações por e-mail após o pagamento.
  • Resolução de erro de timezone em logs de transações.
Versão 2.2.0 - 2024-10-01
Melhorias:
  • Adição de suporte para múltiplos idiomas na interface do usuário.
  • Atualização das dependências do projeto para melhorar compatibilidade e segurança.
Versão 2.1.0 - 2024-09-15
Melhorias:
  • Adição de suporte a notificações via e-mail para clientes após a conclusão do pagamento.
Correções de Bugs:
  • Correção de erro no cálculo de taxas adicionais em transações internacionais.
Versão 2.0.0 - 2024-09-01
Atualização de Versão:
  • Grande atualização com reestruturação do sistema de integração para suportar múltiplos gateways de pagamento.
  • Refatoração completa do código para maior modularidade e facilidade de manutenção.
Correções de Bugs:
  • Correção de erro de comunicação com servidores em redes locais.
Versão 1.2.0 - 2024-08-01
Melhorias:
  • Implementação de relatórios de transações no painel administrativo.
  • Melhorias na compatibilidade com PHP 7.4.
Versão 1.1.0 - 2024-07-15
Melhorias:
  • Adição de um painel de controle para configuração rápida de parâmetros do WebytePay.
  • Otimização no processo de autenticação para maior segurança.
Correções de Bugs:
  • Correção de erro ao registrar transações com valores decimais.
Versão 1.0.0 - 2024-07-01
Lançamento Inicial:
  • Lançamento inicial do WebytePay com funcionalidades básicas de integração com o Vendi Aqui.
  • Primeira versão do gateway de pagamento suportando transações básicas e integração com bancos locais.
